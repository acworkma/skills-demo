# Conversation Starters — Project Intake & Governance Advisor

Use these conversation starters in Copilot Studio or Agent Builder.

## 1. Submit a new project idea

**Title:** Submit a new project idea  
**Text:** I'd like to submit a new analytics project for governance review. Please help me gather the required intake details, classify the request, and produce the full readiness package.

## 2. Classify a reporting request

**Title:** Classify a reporting request  
**Text:** We need a new internal dashboard for weekly sales and service operations. Can you classify the governance tier, identify required reviews, and recommend the right architecture path?

## 3. Evaluate an AI assistant

**Title:** Evaluate an AI assistant  
**Text:** Our team wants to build an AI assistant that summarizes internal project data and answers questions for managers. Please assess the governance tier and tell us what reviews and controls are required.

## 4. Review an agentic workflow

**Title:** Review an agentic workflow  
**Text:** We're proposing an agent that can automatically trigger follow-up tasks and approvals across systems. Please evaluate the governance impact, required controls, and readiness steps.

## 5. Assess a Copilot extension

**Title:** Assess a Copilot extension  
**Text:** We want to create a Microsoft 365 Copilot extension for project intake and governance guidance. Help us classify the request and produce the standard governance readiness package.

## 6. Triage a multi-source analytics solution

**Title:** Triage a multi-source analytics solution  
**Text:** We are combining data from finance, operations, and customer support to support internal decision-making. Please determine the governance tier and list the required reviews and controls.

## 7. Check external sharing risk

**Title:** Check external sharing risk  
**Text:** We plan to generate insights for a partner-facing solution and may share outputs outside the company. Can you evaluate whether this changes the governance tier and approval path?

## 8. Validate project intake completeness

**Title:** Validate project intake completeness  
**Text:** I have a rough project concept but may be missing some required fields. Ask me the clarifying questions you need, then generate the standardized readiness package once the intake is complete.
